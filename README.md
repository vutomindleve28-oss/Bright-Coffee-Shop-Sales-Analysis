# Bright-Coffee-Sales-Analysis
Business Insights for a new CEO using historical transactional data from Bright Coffee Shop
# Problem Statement
Bright Coffee Shop has collected large amounts of transactional data but lacks clear insights into its sales performance. The new CEO needs to understand:

- Which products generate the most revenue
- What time of day are sales the highest
- Which products are underperforming
- How sales trends change over time

Without proper analysis, the company cannot make informed decisions to improve revenue and optimize operations.

# Aim
To analyse historical sales data and provide meaningful insights that will help improve revenue, optimize product performance, and support strategic decision-making at Bright Coffee Shop.

# Objectives
-Analyse total revenue and sales performance
-Identify best-selling and least-selling products
-Determine peak sales hours and time trends
-Understand customer purchasing patterns
-Provide actionable recommendations to improve business performance

# Steps Taken
- Brainstorming using Miro
- Project Planning using Canva
- Data processing using Databricks
- Data Analysis
- Data Visualisation
- Presentation
